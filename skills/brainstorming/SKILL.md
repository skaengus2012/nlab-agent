# Brainstorming

## Purpose
Use this skill when you need to explore multiple solutions or define a new feature from scratch.

## Process
1. **Define the Problem**: Clearly state the objective or pain point.
2. **Divergent Thinking**: Generate at least 3-5 distinct ideas without judging them yet.
3. **Convergent Thinking**: Evaluate each idea based on:
   - Feasibility (can we build it?)
   - Impact (will it solve the problem?)
   - Effort (how long will it take?)
4. **Selection**: Choose the best option or a hybrid approach.

## Guidelines
- Encourage wild ideas initially.
- Look for analogies in other domains (e.g., "How would Spotify solve this navigation problem?").
- Consider the User Experience (UX) first.
