# Example Skill

This is a template for creating new skills in the `nlab-agent` repository.

## How to use
1. Copy this folder and rename it to your skill name.
2. Update the `name` and `description` in the YAML frontmatter.
3. Write your instructions here.

## Guidelines
- Keep instructions concise.
- Use Markdown for formatting.
- Include examples where possible.
