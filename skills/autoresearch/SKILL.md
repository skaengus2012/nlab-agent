# Auto Research

## Purpose
Automate the gathering and synthesis of information for a specific query or topic.

## Process
1. **Define Search Queries**: Generate 3-5 specific search queries based on the main topic.
2. **Execute Search**: Use the search tool to find relevant pages.
3. **Analyze & Extract**: Read the most promising pages and extract key facts, code examples, or data points.
4. **Synthesize**: Combine the findings into a structured report.
5. **Verify**: Check for conflicting information and resolve it if possible.

## Guidelines
- Focus on authoritative sources (official docs, reputable blogs, stack overflow).
- Always include links/sources in the final report.
- If researching a library, prioritize usage examples and potential pitfalls.
